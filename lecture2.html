HTML Attributes:

    An attribute is used to define the characteristics of an HTML element by giving additinal informations.
These informations adjust element'behavior and configuration. 
The attributes are placed inside the element's opening tag.
Attributes usually come in name/value pairs like: name="value".

      [align,alt,background,bgcolor,border,charset,color,height,herf,lang,span,src,target,width,...]
are all attributes used to adjust tagnames.

Attribute Examples:

Align attribute:

           <body>
              < p align="left" paragraph no1   </p>
              < p align="center" paragraph no2 </p>
              < p align="right" paragraph no3  </p>
          </body>

The result will be:

     paragraph no1
                     paragraph no2
                                      paragraph no3

Font and color attributes(more than one attribute can be applied to the same tag):

   <body>
      < p font="20px";color="red"   paragraph no1 </p>
      < p font="25px";color="green" paragraph no1 </p>
      < p font="30px";color="blue"  paragraph no1 </p>
   </body>

The result will be:

     paragraph no1
     paragraph no2              
     paragraph no3

HTML Comments:

     Comment is a piece of code which is ignored by any web browser, but they can help document your HTML source code.
Comments help you and others understand your code and increases code readability.
HTML comments are placed in between <!-- ........ --> tags.

Example

  
 <!DOCTYPE> 
 <html>
   <head>     <!-- The start of head section -->
      <title> Comment Example <title>
   <head>     <!-- The End of head section -->
      <body>    
         <p> paragraph no1   </p>
         <p> paragraph no2   </p>
         <p> paragraph no3   </p>
      </body> 
 </html>

The result will be:

Both comment statements at the start and the end of head section were ignore by the web browser
