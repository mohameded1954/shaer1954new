Lecture Summary

What is HTML:

  • HTML stands for Hyper Text Mark Up Language. And is not programming language.
  • HTML is a text formatting language tells the browser how to display web pages.
  • HTML describes the structure of web page by using HTML elements.
  • HTML elements include headers, prargraphs,buttons,images,tables, lists and other things

Web prowsers:

        Web Browser 
  Is a software program which is used to read source codes and 
  show web pages.                 
  Normally Web Browsers include a browser window with

  settings and other tools.       
        Browser Compatability : 
  refers to the ability of a given Web site to appear fully

HTML Document:

HTML Page Structure:

HTML page consists of two main parts:

First :Head section starts with <head >opening tag.
        and ends with </head>closing tag.     
head section contains information specific to the page including title tag,
       meta tag and links to both style sheets and Script 
Second: body section starts with <body >
opening tag and ends with </body>closing tag.
body section contains the informations
    the client will see including headers,images,buttons 
and paragraphs

case,since it is neat and clean. to declare that the document is an HTML5 document

states the document language which is English

< meta charset = "utf-8" /> is a meta tag to declare that the document
is in utf-8 code (more information about meta tags and utf-8 code will be in next sections)

< title > specifies title for the html page this title will be shown in the browser's title bar

< h1 > -< h6 > tags display headings of different sizes h1 the largest h6 the smallest
