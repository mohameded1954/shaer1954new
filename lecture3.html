HTML Links:

    A link or hyperlink is a connection from one web page to another, the internet is made up of lots of links.
Almost any web content can be converted to a link so that when clicked or otherwise activated
the web browser goes to another web address (URL).
A link has two ends, called anchors. The link starts at the source anchor and points to the
destination anchor, which may be any web resource, for example, an image, an audio or video clip,
a PDF file, an HTML document or an element within the document itself, and so on.
Normally HTML link is colored to show whether its  Unvisited or   visited or   active.  
     Note: A link does not have to a text,It can be an image or any HTML element.

HTML Link Syntax:

 The < a > tag is used to specify HTML link in web pages.
the hyper link will look like:  < a href="URL"> link text < /a>,Where href attribute indicates the required destination.
and "link text" is the visible part to the reader.
     Note:  Search engines use link text to index target files, 
            it is a good idea to include key words in link text to describe what is being linked to.
also "title" attribute can be added to the link to give more useful information.

The target attribte:

The target attribute tells the browser where to open the linked document.The targetattribute has the following values:
  • _self Default. Opens the document in the same window/tab as it was clicked
  • _blank Opens the documet in a new window or tab
  • _parent Opens the document in the parent frame
  • _top Opens the document in the full body of the window
